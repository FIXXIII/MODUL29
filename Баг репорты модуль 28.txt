Баг-репорт 1:

Описание: отсутствует ошибка при вводе неверного пароля
Шаги воспроизведения:

Зайти на сайт https://b2c.passport.rt.ru
Ввести корректный номер телефона в поле телефон
Ввести некорректный пароль в поле пароль
Нажать кнопку "Войти"
Ожидаемый результат: под полем ввода пароля появляется надпись "введен не верный пароль"

Фактический результат: Поле пароль остается пустым, надпись с ошибкой не появилась
Окружение: Windows 10, Chrome Версия 112.0.5615.138 (Официальная сборка), (64 бит)

Баг-репорт 2:

Описание: Не кликабельный логотип "Ростелеком" в шапке сайта
Шаги воспроизведения:

Зайти на сайт https://b2c.passport.rt.ru
Нажать на логотип "Ростелеком" в шапке сайта
Ожидаемый результат: Страница перезагружается. С любой страницы нас переводит на форму авторизации
Фактический результат: Ничего не происходит

Окружение: Windows 10, Chrome Версия 112.0.5615.138 (Официальная сборка), (64 бит)


Баг-репорт 3:

Описание: отсутствует кнопка авторизация по временному паролю
Шаги воспроизведения:

Зайти на сайт https://b2c.passport.rt.ru
Ввести номер телефона в поле телефон
Нажать кнопку войти
Ожидаемый результат: Появляется кнопка "войти по временому паролю" из смс

Фактический результат: При нажатии на кнопку войти ничего не происходит

Окружение: Windows 10, Chrome Версия 112.0.5615.138 (Официальная сборка), (64 бит)
